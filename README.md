# ripgrim

my personal site. a short intro, what i'm building, recent GitHub activity, and
where to find me. from Oregon, way too into open source, mostly shipping things
i'd use anyway.

## stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [Geist](https://vercel.com/font)
- [cuelume](https://cuelume-site.pages.dev) for interaction sounds

## develop

```bash
npm install
npm run dev
```

then open [localhost:3000](http://localhost:3000).

## project

- [tripwire](https://tripwire.sh): a firewall for your repo. it checks every
  contribution so you keep the good stuff and drop the rest.
